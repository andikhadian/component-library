## üöÄ Getting Started

Install the package using npm or yarn

## ‚òï Usage

```sh
  import { Button } from "@andikhadian/component-library";

  const App = () => {
    return (
      <div>
        <Button>Hello Button</Button>
      </div>
    );
  };

  export default App;
```
